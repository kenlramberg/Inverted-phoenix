# Company Partnership & Outreach System

## Overview

AU4A has **NO traditional ads**. Instead, companies can get exposure by **donating products** to AU4A members.

### The Model

**What Companies Provide:**
1. Products donated to AU4A members who need them
2. Logo + rights to display on AU4A homepage
3. Direct shipping to members
4. Full warranty on donated products

**What Companies Receive:**
1. Logo displayed at bottom of AU4A homepage
2. Association with "greatest social experiment in history"
3. Brand exposure tied to generosity (not advertising)
4. Recognition as a company that gives, not just sells
5. Part of internet history

**What Is NOT Included:**
- ❌ Product listings on site (only logos)
- ❌ Paid placement or ranking
- ❌ Sponsored content
- ❌ Traditional advertising
- ❌ Affiliate commissions

## API Endpoints

### Company Sponsors

**Create Sponsor**
```bash
POST /api/sponsor
{
  "company_name": "TechCorp",
  "website": "https://techcorp.com",
  "logo_url": "https://logo-url.com/logo.png",
  "contact_email": "partnerships@techcorp.com"
}
```

**Get Sponsors (for homepage)**
```bash
GET /api/sponsors?active_only=true&for_display=true
```

**Activate Sponsor**
```bash
PATCH /api/sponsor/{sponsor_id}/activate
```

### Donated Products

**Add Product**
```bash
POST /api/donated-product
{
  "sponsor_id": "sponsor_uuid",
  "product_name": "Laptop",
  "description": "High-performance laptop",
  "quantity_available": 10
}
```

**Get Products**
```bash
GET /api/donated-products?available_only=true
```

**Match Products to Request**
```bash
GET /api/match-products/{request_id}
```

### Outreach Engine

**Create Outreach Campaign**
```bash
POST /api/outreach
{
  "company_name": "Example Tech",
  "company_email": "partnerships@example.com",
  "company_website": "https://example.com",
  "notes": "Good fit for AU4A"
}
```

This automatically generates:
- Email subject
- Complete email body with pitch
- Stores campaign for tracking

**Get Outreach Campaigns**
```bash
GET /api/outreach?status=pending
```

**Mark as Sent**
```bash
PATCH /api/outreach/{campaign_id}/mark-sent
```

## Workflow

### 1. Outreach
1. Create outreach campaign via `/api/outreach`
2. System generates email with pitch
3. Send email manually (or integrate email service)
4. Track status (sent, opened, responded)

### 2. Partnership Agreement
1. Company responds positively
2. Create sponsor via `/api/sponsor`
3. Agree on terms:
   - What products to donate
   - Shipping arrangements
   - Warranty terms
4. Upload logo

### 3. Activation
1. Company signs agreement
2. Activate sponsor via `/api/sponsor/{id}/activate`
3. Logo automatically displays on homepage
4. Add donated products to catalog

### 4. Fulfillment
1. Member submits request
2. System matches with donated products
3. Coordinator facilitates connection
4. Company ships directly to member
5. Member confirms receipt
6. Product marked as fulfilled

## Important Rules

1. **NO product listings on homepage** - Only logos
2. **NO paid placement** - All equal exposure

4. **Must ship + warranty** - Non-negotiable
